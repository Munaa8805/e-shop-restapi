Body Parameters:

Reset password using token
Body Parameters:
token (string) required - Reset token from email
password (string) required - Minimum 4 characters
PATCH /auth/update-me Auth Required
Update current user profile
Body Parameters (all optional):
name (string) - Minimum 4 characters
email (string) - Valid email format

GET /products Admin
Get all products with populated category, brand, company, and creator
GET /products/:id Admin
Get product by ID with populated fields
POST /products Admin
Create a new product
Body Parameters:
name (string) required
price (number) required - Must be non-negative
category (ObjectId) required - Category ID
createdBy (ObjectId) required - User ID
description (string) - Optional
image (string) - Optional image URL
quantity (number) - Optional, default 0
brand (ObjectId) - Optional brand ID
company (ObjectId) - Optional company ID

Update product by ID
DELETE /products/:id Admin

PUT /products/:id/upload-image Admin
Upload single product image (resized to max 1200x1200px)
Form Data:
image (file) required - JPEG, PNG, GIF, or WebP (max 5MB)
PUT /products/:id/images Admin
Upload multiple product images (up to 10, resized to max 1200x1200px)
Form Data:
images (file[]) required - Array of images (JPEG, PNG, GIF, or WebP, max 5MB each)
?replace=true - Optional query param to replace all existing images instead of appending
